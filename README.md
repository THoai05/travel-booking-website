# Booking Hotel Website

---

## Cài đặt & Chạy dự án

```bash
# Clone project
git clone <repo-url>
cd travel-booking-website

# Cài đặt BE và Chạy
cd BookingHotel-be
npm install

npm run start:dev
# => http://localhost:8080

# Cài đặt FE
cd ../BookingHotel-fe
npm install
npm run dev
# => http://localhost:3000
